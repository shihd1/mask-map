台中口罩地圖
口罩地圖：成人 口罩地圖：兒童 藥局查詢 設計動機

Just like many other Taiwanese families, during the Corona Virus crisis, our family was in dire need of masks to protect ourselves. However, the many stores we went to had no masks, so we didn't know where to get some. To solve this problem, I created a map using Javascript and Java that shows all the locations where you could get masks along with updated information about how many masks the store currently has. The map is centered around Taichung because it is where I live.
